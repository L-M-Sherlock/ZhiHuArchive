问题描述

1.supermemo里，我点几次跟笑脸并排的那几个图案（包括表示我并没有学好的），卡片在当时就没了，但是有的时候并没有掌握，想要像anki一样可以在当时重复出现，（并且是以问答题的形式，而非答案和问题都在同一页面）以加强记忆，应付可能一两个小时之后的考试。

2.当我添加材料后，如何在当时立刻学习，而非等到第2天。可以是不影响supermemo本身的记忆算法的另外的练习。

既然 Daniel 写了第一点，我就答一下第二点吧~

少量材料

直接 Ctrl + Shift + J，输入你想要把新材料插入到待办队列的位置：

这里输入 47 就是队尾，1 就是队首。

大量材料

  1. 先在知识树里右键点击你所添加的材料所属的 Branch 的根节点（如图）

2. 点击 Process branch，找到 Learning - Add all to outstanding 并点击：

3. 输入一个数字 n，代表在原有的队列基础上，每隔 n 个卡片插入一个 branch 中的卡片。

以上就是如何将材料立即加入当天待办队列的方法，更多有关筛选组学习的技巧，请参考：
