许多人认为记忆对深刻的创造性工作不重要

@Thoughts Memo

学校≠教育≠技能；文凭溢价=80%信号传递+20%人力资本

发表于 2022年01月07日

32 👍 / 0 💬

当告诉别人间隔重复记忆系统让记忆成为一种选择[1]时，人们往往反应相当消极：「背诵知识有什么意义？死知识才不是关键。我想要得到概念性的理解、创造力、艺术性，诸如此类！」

对此的一个回应是指出事实上间隔重复记忆系统可以用来发展概念理解[2]。但记忆知识没有价值的说法也必须认真对待。在此我有些回应：

  • 基本认知任务依赖于记忆
  • 一些创新洞察依赖于长期记忆[3]
  • 深刻的理解要求细致掌握基础知识[4]
  • 专业水准需要建立复杂的组块重编码方案[5]
  • 人们在学校经受过死记硬背，因由这些负面经历而低估记忆的重要性[6]
  • 间隔重复记忆系统让记忆更加自动化，从而有助于更深入的琢磨思考[7]
  • 间隔重复记忆系统可以缩短科目学习中不愉快的早期阶段[8]

参考文献

Matuschak, A., & Nielsen, M. (2019, October 0). How can we develop transformative tools for thought?[9] https://numinous.productions/ttft (see section “How important is memory anyway?”)
