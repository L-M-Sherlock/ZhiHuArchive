有批量删除topic但是保留这些topic下的item的方法吗?

简单，用好 browser 即可。

首先，打开知识树界面，对你要操作的 branch 右键（想操作全部就右键根节点），然后点 View - Branch（这步能查看这个 branch 下全部卡片）

然后，我们到了 subset element，点击工具栏里紫色的 T，可以筛选出里面的 topic

这样一来剩下的就都是 topic 了，我们右键打开菜单，进行 Process browser

点击 Learning - Dismiss（或者 Done），就可以批量暂停这些 topic 的复习了。
