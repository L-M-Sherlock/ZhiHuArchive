如何在 SuperMemo 中复习/学习指定的folder/branch？

本系列专门介绍 SuperMemo 的各种小技巧和绕坑方法：

分两种情况

一种是记忆过的问答卡片和摘录卡片，如：

一种是待记忆的新材料，如：

针对第一种情况

针对第一种材料，只需要在知识树里右键对应的 branch，然后选择菜单里的 learn 即可：

其复习进度会显示在 statistics 面板中的 review 里

多按几次 esc，然后按回车或者 next repetition，然后 SuperMemo 会询问是否继续进行筛选组学习，点 no 就可以退出筛选组学习。

遗憾的是，这种方法不适用于新材料，需要「曲线救国」

针对第二种情况

学习等待队列的卡片需要将 stages 切换到 New material，操作如图所示

由于等待队列的卡片是有默认顺序的，我们只需要通过设置这个顺序，就能实现优先学习某一个 folder 或者 topic，具体的方法如下：

首先，先打开等待队列，View->Other->Pending

然后，任意处右键->Sort->By tree，就可以对 pending 队列里的卡片按照知识树顺序排序


最后右键->Tools->Save pending，就能保存排序结果。


什么？你问我这哪门子实现了学习指定 folder 了？

。。。

在知识树上，把想要先学的 folder 移到其他 folder之前，然后按上面一通操作，然后进入 new material 学习就可以先学这个 folder 了。虽然有些变扭，但是逻辑上符合学习的思路。

知识树类似一本书的目录，通过上述排序，就可以先学这本书前面的内容。
