介绍一下这款插件：

Deadline2

An update of the Anki Deadline addon with GUI configuration.

具有GUI配置的 Anki Deadline 插件更新。

This addon is used to automate the process of adjusting the new cards per day setting for an options group. The setting is calculated by finding (number of new cards left/number of days until deadline). Both the options group and new cards per day setting is automatically managed by the addon.

此插件用于自动调整选项组每天新卡设置。 通过查找（剩余的新卡数量/截止日期之前的天数）来计算设置。 插件会自动管理选项组和每天的新卡设置。

To use:

1. Go to the main deck browser screen

2. Click the Tools Menu -> Deadline -> Configure Deadlines

3. Click the Add button on the right

4. Select a deck, then select a deadline when you want to have all new cards for the deck seen by.

Note: Only decks that have cards in them will be shown.

5. Click OK

Note:Repeat 3-5 for each deck you would like to add a deadline to

6. Click Process Deadlines

使用方法：
1. 到主页面
2. 点击工具菜单 -> Deadline -> Configure Deadlines
3. 点击右侧添加按钮
4. 选择一个牌组，然后选择一个 Deadline。
5. 点击 OK
6. 点击处理 Deadlines
