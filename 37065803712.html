在错误的算式“1 + 2*3 + 4*5 + 6*7 + 8*9 = 479”中，添加一对括号使其成立，可以将括号放在“6*7 + 8”处，具体修改为：

```
1 + 2*3 + 4*5 + 6*(7 + 8)*9 = 479
```

这样计算顺序变为：

1. 计算括号内：7 + 8 = 15  
2. 然后进行乘法：6*15 = 90  
3. 接着计算：90*9 = 810  
4. 最后进行加法：1 + 6 + 20 + 810 = 479

因此，等式成立。
