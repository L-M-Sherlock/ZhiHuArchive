基于石墨文档的 Anki 学习小组，内容较为复杂，熟练使用 Anki 的同学可以参考，具体细节待完善。


阅读须知：本文内容主要面向有一定经验的 Anki 学习者，且身边(网上)也有使用 Anki 的同学。


核心思想：利用石墨文档的在线编辑和划词评论功能，实现 Anki 卡片的内部分享与相互讨论。

合作流程：

  1. 个人牌组使用 Basic (基础) 和 Cloze (填空)两种类型制卡
  2. 定期导出为 纯文本格式笔记 并复制进自己的石墨文档表格( Anki 插件：1983204951，可以导出在卡片浏览器中选中的卡片，节省时间)
  3. 浏览同学的新卡片，挑选自己钟意的卡片，讨论有争议的卡片
  4. 在线共同编辑公共牌组，每位组员承包自己力所能及的牌组，可以将自己的卡片整合入公共牌组
  1. 公共牌组每完成一个模块，就进行共同审核，最后制作成 编号-问题-答案-分类 和 编号-填空-说明-分类 模板的牌组(点击模板下载)

小技巧：

  • 使用 加粗 斜体 下划线 来进行强调，少使用颜色标记(提高导出的文本格式笔记的可读性)
  • 定期导出可以使用插件，用 added:7 查找 7 天内添加的新卡片，然后直接在浏览器中导出
  • 分类字段制卡时就填好可以节省很多后期整理的时间
  • 由于表格制卡不区分填空题与问答题，导入时全部使用问答题模板，然后搜索 {{c 找出填空题，再将填空题卡片类型改正
  • 图片、音频等媒体文件，可以用 Github 或云盘同步。

好处：

  1. 通过分配制卡项目，减少每个人的重复制卡，能够节省大量制卡时间
  2. 用石墨表格分享卡片，可以让自己的卡片受到检验，得到与同学交流的机会，在交流中学习
  3. 浏览同学的分享卡片，可以开拓自己的学习视野，发现自己的学习盲区，还能意外得到自己需要的卡片

缺陷：

  1. 没有强制力，分享和检验都属于个人自愿，较理想主义，适合小团体内部合作
  2. Anki 与石墨表格直接的导入导出环节较多，需要占用部分认知资源来坚持
  3. 部分操作需要使用电脑，部分学生没有使用电脑的机会
