继续翻译官方教程：Quickly Organizing Hierarchies

我会尽可能用自己的例子，方便大家理解

学习快速导航和组织你的知识库对于减少使用 RemNote 作为扩展大脑时的摩擦至关重要。这里有一些快捷方式，可以使导航更加流畅和直观。

快速移动 Rem

拖放

在编辑器中，你可以按住任意一个 ● 符号并将其拖动到其他地方。此外，你可以先选择多个 Rem，然后挑选任何 ● 来同时移动它们：

Ctrl + Shift + M

你可以通过单击 Rem 的文本，然后按 Ctrl + Shift + M 来快速地将 Rem 移到新位置。一个菜单将出现，为 Rem 选择新的父级。 选择父级后，将在旧的 Rem 位置添加一个入口，以显示其新位置。 请随意删除该入口； 底层的 Rem 不会受到影响，因为它现在在另一个文档中。（可以通过单击顶部的浅蓝色栏并按退格键来删除入口。）

剪切 / 粘贴

你可以按Shift + ↑ / ↓ 选择多个 Rem，然后按 Ctrl + X 剪切它们。 使用方向键导航到其他地方，然后按 Ctrl + V粘贴。

Alt + ↑ / ↓

按Alt + ↑ / ↓ 将向上或向下移动当前 Rem。 这对于快速操作很有用。

将 Rem 复制为引用、标签或入口

在组织你的想法时，你会经常想要在已有的想法之间建立联系。在 RemNote 中有三种方法可以做到这一点：使用 Rem 引用、标签和入口。

  • 使用 Ctrl + Shift + R 快速复制 Rem 引用。
  • 使用 Ctrl + Shift + S 快速复制 Rem 标签。
  • 使用 Ctrl + Shift + P 快速复制 Rem 入口。
